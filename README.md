# Drive
Drive — это небольшой учебный проект на C++ и SFML. В игре нужно управлять машиной, уворачиваться от препятствий и набирать очки. Реализованы жизни, таймер, счет и простой интерфейс. Проект сделан для практики работы с графикой, обработкой ввода и базовой игровой логикой.
